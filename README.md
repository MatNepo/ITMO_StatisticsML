# Statistics and Machine Learning Labs

Here you can find labs on Statistics and Machine Learning from ITMO course

## Structure
```
-- current page
    |-- lab_1 - Вероятностное пространство, формула Байеса
    |-- lab_2 - Оценки математического ожидания, дисперсии, медианы
```

Каждая лабораторная работа содержит Jupyter notebook с решениями и соответствующие материалы задания.
